# Orbital Chaos — GPU Training

Train ML models for spacecraft orbit prediction using NASA SSC data.

- **Dataset**: [datamatters24/orbital-chaos-nasa-ssc](https://huggingface.co/datasets/datamatters24/orbital-chaos-nasa-ssc)
- **Models**: LSTM, Transformer (encoder-only direct), Multi-modal (orbit + solar wind)
- **Spacecraft**: ISS (LEO), DSCOVR (L1), MMS-1 (HEO)
